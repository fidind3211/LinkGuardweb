Bot Commands
/check

Permission: none
Function: checks a user against our database

/clean run

Permission: ban members
Function: removes all leakers from the server.

/clean view

Permission: manage messages
Function: view who would be removed if you ran a clean

/list view

Permission: none
Function: view a list of the leakers that we ban

/list request

Permission: none
Function: request an addition to this list

/list appeal

Permission: none
Function: appeals an addition to the list. we probably won't consider it.
